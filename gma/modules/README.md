# GMA Module Mounts

This folder contains reusable GMA module repositories mounted as Git submodules. AccessControl is currently the source-first local package exception until its `GMA-Module-AccessControl` repository is created. In GitHub, open module documentation through the owning source repository instead of a deep skeleton path under `gma/modules/...`.
